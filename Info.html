Mark up

HTML5

웹표준을 지키며 기본적인 마크업을 구성합니다. 단순히 잘 보이기 위해 코드를 짜는게 아닌, 세부적인 구조를 짜려 노력합니다. 용도에 맞는 태그를 쓰고 있습니다.

CSS3

구조에 따른 스타일링을 구현하고있습니다. 반응형을 위해 뷰포트 기준으로 단위를 지정하며, 상황에 맞는 display를 선택해 레이아웃을 구현합니다.

Lauguage

JS

가장 유동적이며, 언어 중에서 제일 재미있다고 느끼는 부분입니다. 타입에 크게 연연하지 않는다는 점이 접근하기 쉬운 것 같습니다. 오픈소스를 활용해 저의 것으로 만드는 부분에 가장 강합니다.

C

컴퓨터 언어의 기본이라 생각합니다. C를 통해서 컴퓨터의 알고리즘, 내부구조를 알게되었으며 다른 언어를 사용 시 큰 도움이 되었습니다.

C#

유니티를 사용해 스크립트를 짤 때 사용합니다. 기본 문법을 사용해 간단한 게임을 제작했으며 VR형 게임과 모션센서를 통해 사용자와의 동적 움직임을 사용한 프로그램을 제작했습니다.

Java

언어의 기본적 문법을 배우면서 백엔드 영역을 이해하고, 프론트의 알고리즘을 알게되면서 저의 개발공부에 가장 도움이 많이 되었습니다.

php

서버와 통신하기 위해 사용한 언어입니다. 기본적으로 html과 비슷해 언어를 사용하는데에서는 어려움이 없었으나, 데이터베이스를 주고받는 과정에서는 어려움이 느껴졌습니다.

Library

React

React 프로젝트를 진행하며 상태관리주기, 컴포넌트에 대한 이해도가 있습니다. 모듈을 분리하는 법을 배웠으며 html 없이 웹페이지를 제작할 수 있습니다.

Node.js

컴퓨터 언어의 기본이라 생각합니다. C를 통해서 컴퓨터의 알고리즘, 내부구조를 알게되었으며 다른 언어를 사용 시 큰 도움이 되었습니다.

JQuery

유니티를 사용해 스크립트를 짤 때 사용합니다. 기본 문법을 사용해 간단한 게임을 제작했으며 VR형 게임과 모션센서를 통해 사용자와의 동적 움직임을 사용한 프로그램을 제작했습니다.

three.js

3D공간을 구성할 수 있으며 오직 코드로만 공간을 제작할 수 있습니다. 컨셉에 맞는 컨트롤을 선택해 제작할 수 있습니다. 오브젝트를 로드할 수 있으며, 오브젝트에 물리엔진 적용이 가능합니다.

DB

mySQL

서버에 데이터베이스를 생성하고, 읽고, 사용하고, 삭제할 수 있습니다.

CoWork

Git

커맨드를 통한 프로젝트 빌드가 가능하며 프로젝트를 서버에서 관리를 할 수 있습니다. git을 사용해 다른 개발자와 협업이 가능합니다. 커맨드를 통해 repository생성, branch생성,관리를 할 수 있습니다. github에 공개된 소스들을 활용해 더 많은 기능제작을 합니다.

Github

Figma
